## Quick Start Guide

1. **Clone/Download** this code structure

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the application:**
   ```bash
   npm run start:dev
   ```

4. **Seed sample data:**
   ```bash
   npm run seed
   ```

5. **Access Swagger Documentation:**
   Open http://localhost:3000/api in your browser

6. **Run tests:**
   ```bash
   npm run test
   npm run test:e2e
   ```
